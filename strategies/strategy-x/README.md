# Strategy X

This folder is reserved for the Strategy X Pine Script strategy.

## Files

- `strategy-x.pine` - strategy source code

## Installation

1. Open TradingView and go to **Pine Editor**.
2. Copy the contents of `strategy-x.pine`.
3. Paste the code into Pine Editor.
4. Click **Save**.
5. Click **Add to chart**.
